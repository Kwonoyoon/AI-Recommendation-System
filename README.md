# AI-Recommendation-System
# 🎬 AI 기반 영화 추천 시스템
사용자가 입력한 영화와 유사한 영화들을 TF-IDF + Cosine Similarity 기반으로 추천하는 간단한 콘텐츠 기반 추천 모델입니다.

## 📚 사용 기술
- Python, Pandas, Scikit-learn
- TF-IDF 벡터화, Cosine Similarity

## 📁 데이터셋
- Netflix Popular Movies Dataset (`n_movies.csv`)

## 💡 주요 기능
- 영화 제목 입력 → 비슷한 영화 5개 추천
- 중복 제목 처리 및 결측치 제거
